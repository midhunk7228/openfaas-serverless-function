=====================================
YOUR DEPLOYMENT STEPS:
=====================================

□ Step 1: Push code to GitHub
  Command: git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
  Command: git push -u origin main

□ Step 2: Go to render.com and sign up with GitHub

□ Step 3: Click "New +" → "Web Service"

□ Step 4: Connect your GitHub repository

□ Step 5: Fill in the settings above

□ Step 6: Click "Create Web Service"

□ Step 7: Wait 2-5 minutes for deployment

□ Step 8: Get your URL (e.g., https://brands-list-api.onrender.com)

□ Step 9: Test your API!

=====================================

=====================================

# Replace with your actual Render URL:
export API_URL="https://brands-list-api.onrender.com"
